# Bcrypt

An exercise of signup and login with Bcrypt. BackEnd proyect with Node/Express, generated with express-generator, ejs, mongoDB and Mongoose.

## Packages

- bcrypt
- body-parser
- cookie-parser
- debug
- ejs
- express
- mongoose
- morgan
- serve-favicon

# Download
You can clone it, or download.
Then
`npm install`
`npm start`
With mongoDB running!!
Then open at
`localhost://3000`
